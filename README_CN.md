### 1. [基于验证器的数学推理难题生成方法](http://arxiv.org/abs/2605.06660v1)
- **摘要**: 大型语言模型在解决科学和数学问题方面展现出强大能力，但在生成有效、具有挑战性且新颖的问题方面仍存在不足——而这一能力对于推进大模型训练和实现自主科学研究至关重要。现有问题生成方法要么依赖昂贵的人类专家参与，要么采用简单的自我博弈范式，这种范式常因奖励机制漏洞而产生无效问题。本研究提出VHG框架，这是一种基于三方自我博弈的验证器增强型难题生成框架。通过在传统的出题者-解题者二元结构中引入独立验证器，我们的设计将出题者的奖励约束为问题有效性（由验证器评估）与难度（由解题者评估）的联合决定。我们实现了两种验证器变体：基于符号系统的硬验证器和基于大语言模型的软验证器，并在不定积分任务和通用数学推理任务上进行了评估。实验结果表明，VHG在所有基线方法中均展现出显著优势。

### 2. [当无基准存在时：在没有真实标签的情况下验证大语言模型安全评分的比较性](http://arxiv.org/abs/2605.06652v1)
- **摘要**: 许多部署场景在缺乏针对特定语言、行业或监管体系的标注基准时，仍需对候选语言模型进行安全性比较。我们将这一场景形式化为"无基准比较安全评分"，并明确了基于场景的审计可作为部署证据的契约条件。评分仅在固定场景包、评分标准、审计方、评审方、采样配置及重跑预算下有效。由于缺乏标注数据，我们以工具有效性链替代真实标签一致性：该链条包含对受控安全与消融消解对比的响应性、目标驱动方差对审计方与评审方人为因素的支配性，以及跨重跑稳定性。我们在本地优先评分工具SimpleAudit中实例化该链条，并在挪威语安全包上完成验证。安全与消融消解目标的AUROC值介于0.89至1.00之间，目标身份是主导方差成分（η²≈0.52），严重性分布曲线在十次重跑后趋于稳定。将该链条应用于Petri工具时，显示其兼容两种工具。实质性差异出现在链条上游的声明契约执行与部署适配环节。通过挪威公共部门采购案例（对比Borealis与Gemma 3）展示实际证据：更安全的模型取决于场景类别与风险度量指标。因此，评分、匹配差值、临界比率、不确定性以及所使用的审计方与评审方必须联合报告，而非简化为单一排名。

### 3. [超越负向展开：基于隐式负梯度的纯正向策略优化](http://arxiv.org/abs/2605.06650v1)
- **摘要**: 基于可验证奖励的强化学习（RLVR），凭借其确定性验证机制，已成为提升大语言模型推理能力的主流范式。学界见证了从近端策略优化（PPO）到分组相对策略优化（GRPO）的快速演进，其中GRPO通过分组正负轨迹的简单估计替代了复杂的优势函数计算。然而我们注意到，负轨迹可能缺乏失败严重程度的梯度划分，且组合爆炸问题使得在稀疏二元奖励下惩罚少量采样负轨迹难以覆盖有意义的奖励信号。本文提出正向策略优化（POPO），这是一种仅通过在线正轨迹即可实现学习的创新RLVR框架。具体而言，POPO在正轨迹集上采用有界重要性采样，因此无需使用分离的负轨迹进行梯度引导。我们证明通过轨迹重分配强化正概率时，隐式负梯度会自然涌现。其次，POPO通过两种机制稳定策略优化：首先采用具有动量自适应律的孪生策略网络实现稳定策略演化；其次在孪生表征空间中使用有界相似性惩罚项替代KL散度。我们使用公开成熟的文本大语言模型（如Qwen系列）在各级数学基准上开展广泛实验。实验表明POPO能达到与GRPO相当甚至更优的性能，值得注意的是，基于Qwen-Math-7B的POPO在AIME 2025上取得36.67%的准确率，显著超越GRPO的30.00%。消融实验与参数扫描研究进一步验证了POPO各组件的必要性与鲁棒性。
