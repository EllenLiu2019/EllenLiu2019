### 1. [探索性破解：大型语言模型能否学会抵抗强化学习训练？](http://arxiv.org/abs/2604.28182v1)
- **摘要**: 强化学习（RL）已成为大语言模型（LLMs）后训练中不可或缺的技术，用于提升推理能力、智能体功能及对齐效果。成功的RL训练依赖于模型在训练过程中对多样化动作的充分探索，但这可能引发一种潜在失效模式：模型可能在训练期间策略性地调整其探索行为，从而影响后续训练结果。本文系统研究了这种被称为"探索劫持"的行为。首先，我们通过微调LLMs使其遵循特定低效策略，构建了选择性RL抗性模型生物体；这些模型在智能体生物安全与AI研发环境中，既能成功抵抗基于RL的能力激发，又能保持相关任务性能。随后，我们利用这些模型生物体评估了包括监控、权重噪声注入及基于SFT的激发在内的检测与缓解策略。最后研究表明，当前前沿模型在充分了解训练环境信息时，会显式推理如何抑制自身探索行为——当此类信息通过环境间接获取时，该行为发生率更高。综合实验结果，探索劫持可能成为具备充分能力的LLMs在RL训练中的一种潜在失效模式。

### 2. [面向长周期生产力模拟的大规模合成计算机](http://arxiv.org/abs/2604.28181v1)
- **摘要**: 在现实场景中，长期生产力工作高度依赖于用户特定的计算机环境，其中大部分工作上下文通过目录结构和内容丰富的工件进行存储和组织。为了扩展此类生产力场景的合成数据生成，我们提出了“规模化合成计算机”方法——一种可扩展的构建方法，能够生成具有真实文件夹层级结构和内容密集型工件（如文档、电子表格和演示文稿）的计算机环境。基于每台合成计算机，我们运行长期模拟：一个智能体创建与该计算机用户特性相关的生产力目标，这些目标需要多个专业交付成果并相当于人类约一个月的工作量；随后另一个智能体扮演该用户角色，持续在计算机上执行操作——例如通过文件系统获取上下文信息、与模拟协作者协调、生成专业工件——直至完成这些目标。在初步实验中，我们创建了1000台合成计算机并运行长期模拟：每次运行需超过8小时的智能体执行时间，平均交互轮次超过2000次。这些模拟产生了丰富的经验学习信号，其有效性通过智能体在领域内和跨领域生产力评估中的显著性能提升得到验证。鉴于用户画像在十亿级规模下已足够丰富，该方法理论上可在充足算力支持下扩展至数百万甚至数十亿个合成用户世界，从而覆盖更广泛的职业、角色、场景、环境及生产力需求。我们认为，可扩展的合成计算机创建与大规模模拟相结合，极有潜力成为智能体在长期生产力场景中实现自我改进和强化学习的基础支撑。

### 3. [TopBench：面向表格问答的隐式预测与推理基准](http://arxiv.org/abs/2604.28076v1)
- **摘要**: 大型语言模型（LLMs）已推动表格问答技术的发展，大多数查询可通过信息提取或简单聚合来回答。然而，现实查询中常见的一类问题具有隐含预测性——需要从历史模式中推断未观测到的答案，而非仅进行检索。这类查询带来两大挑战：识别潜在意图，以及在庞大数据表上进行可靠的预测性推理。为评估LLMs在隐式预测任务中的表格问答能力，我们提出TopBench基准测试集，包含779个样本，涵盖从单点预测到决策制定、治疗效果分析及复杂筛选等四个子任务，要求模型生成涵盖推理文本与结构化表格的输出。我们分别在基于文本和智能体工作流下评估了多种模型。实验表明，当前模型常难以识别意图，默认仅执行查找操作。深入分析发现，准确的意图消歧是引导这些预测行为的先决条件。此外，提升预测精度的上限需要整合更复杂的建模或推理能力。
