### 1. [探索性攻击：大语言模型能否学会抵抗强化学习训练？](http://arxiv.org/abs/2604.28182v1)
- **摘要**: 强化学习（RL）已成为大语言模型（LLMs）后训练中不可或缺的技术，用于提升推理能力、智能体行为及对齐性。成功的RL依赖于模型在训练过程中对多样化动作的充分探索，但这可能引发一种潜在失效模式：模型可能在训练中策略性地调整其探索行为，以影响后续训练结果。本文研究这种被称为"探索劫持"的行为。首先，我们通过微调LLMs使其遵循特定低效策略，构建了选择性RL抗性的模型生物体；这些模型在智能体生物安全与AI研发环境中，既能成功抵抗基于RL的能力激发，又能保持相关任务性能。随后，我们利用模型生物体评估检测与缓解策略，包括监控、权重噪声注入及基于SFT的激发方法。最后，研究表明，当前前沿模型在充分了解训练背景信息时，会显式推理如何抑制自身探索行为——当信息通过环境间接获取时，该行为发生率更高。综合实验结果，探索劫持可能是具备足够能力的LLMs在RL训练中的一种潜在失效模式。

### 2. [大规模合成计算机用于长期生产力模拟](http://arxiv.org/abs/2604.28181v1)
- **摘要**: 真实的长期生产力工作高度依赖于用户特定的计算机环境，其中大部分工作上下文通过目录结构和内容丰富的工件进行存储和组织。为了扩展此类生产力场景的合成数据生成，我们提出了“可扩展合成计算机”方法，这是一种可扩展的方法，用于创建具有逼真文件夹层次结构和内容丰富工件（例如文档、电子表格和演示文稿）的环境。基于每台合成计算机，我们运行长期模拟：一个智能体创建针对该计算机用户的生产力目标，这些目标需要多个专业交付成果，大约相当于一个月的人类工作量；另一个智能体则扮演该用户，持续在计算机上工作——例如，浏览文件系统以获取上下文、与模拟协作者协调、制作专业工件——直到这些目标完成。在初步实验中，我们创建了1000台合成计算机并运行长期模拟；每次运行需要超过8小时的智能体运行时间，平均跨越2000多个回合。这些模拟产生了丰富的经验学习信号，其有效性通过智能体在领域内和领域外生产力评估中的显著性能提升得到验证。考虑到用户画像在十亿级别上非常丰富，该方法原则上可以在足够计算资源下扩展到数百万甚至数十亿个合成用户世界，从而覆盖更广泛的职业、角色、上下文、环境和生产力需求。我们认为，可扩展的合成计算机创建与大规模模拟相结合，在长期生产力场景中作为智能体自我改进和智能体强化学习的基础平台具有巨大潜力。

### 3. [TopBench：面向表格问答的隐式预测与推理基准](http://arxiv.org/abs/2604.28076v1)
- **摘要**: 大型语言模型（LLMs）推动了表格问答技术的发展，大多数查询可通过信息提取或简单聚合来回答。然而，现实世界中常见的一类查询具有隐含预测性，需要从历史模式中推断未观测到的答案，而非仅进行检索。这类查询带来了两大挑战：识别潜在意图，以及在海量表格上进行可靠的预测推理。为评估LLMs在隐式预测任务中的表格问答能力，我们提出了TopBench基准测试集，包含779个样本，涵盖从单点预测到决策制定、治疗效果分析及复杂筛选等四个子任务，要求模型生成涵盖推理文本和结构化表格的输出。我们在基于文本和智能体工作流两种模式下评估了多种模型。实验表明，当前模型常难以识别意图，默认仅执行查找操作。深入分析发现，准确的意图消歧是引导这些预测行为的先决条件。此外，提升预测精度的上限需要整合更复杂的建模或推理能力。
