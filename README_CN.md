### 1. [UniClawBench：面向真实世界任务的主动式智能体通用基准](http://arxiv.org/abs/2607.08768v1)
- **摘要**: 大型语言模型与多模态大语言模型的快速发展，加速了能够操作日常工具并在真实环境中辅助用户的主动型智能体的涌现。然而现有基准测试难以有效评估此类智能体，因其常依赖沙盒化环境与单轮评估范式。此外，基于场景的任务分类体系将多种模型能力混杂在同一任务类别中，导致难以定位智能体失效的根本原因。为突破这些局限，我们提出UniClawBench——首个面向动态真实场景、以能力为导向的主动型智能体评估基准。该基准围绕五项基础模型能力构建：技能运用、探索能力、长上下文推理、多模态理解及跨平台协调。基于这些能力维度，我们设计了400项双语真实世界任务。与依赖静态预录答案的传统基准不同，本基准通过细粒度的逐步骤完成检查点，在实时Docker容器中评估智能体。我们进一步设计了闭环评估策略，包含执行智能体、隐式监督智能体与用户智能体三个角色，在避免泄露评分标准的前提下模拟真实多轮人类反馈。为解耦基础模型能力与框架级设计选择，我们在多种智能体框架下评估了当前最优模型。通过模型与框架间的全面对比，揭示了基础模型能力与智能体框架设计如何共同塑造真实环境中的表现。为促进后续研究，我们已在https://github.com/HKU-MMLab/UniClawBench 公开基准测试与代码。

### 2. [LLM作为数据标注者的有效性：AMALIA的权威性视角](http://arxiv.org/abs/2607.08731v1)
- **摘要**: 国家语言模型为语言社群提供了一种衡量公民言论与价值的工具。葡萄牙的AMALIA——一个由公共资金支持的90亿参数欧洲葡萄牙语模型——仅在一致性上便展现出竞争力：当被要求对"权威"这一道德基础进行编码时，其与经过训练的人类编码者的一致性，在F1分数上仅比规模大八到十三倍的开源模型低六个百分点。然而，一致性代表的是可靠性，而非有效性。对于需要推断而非从表面特征读取的理论构念，关键在于模型是遵循该构念的理论框架，还是通过相关捷径得出正确编码。我们通过"恢复差距"来检验这一点：当整体提示被分解为编码手册的原子条款，再通过理论的显式规则重新组合时，模型性能的损失程度。若校准能弥合这一差距，则模型应具备跨模型与跨语言的迁移能力；若无法弥合，则构念-模型工具本身很可能是失败根源。我们探究经过校准的英语工具能否迁移至AMALIA-9B及欧洲葡萄牙语。针对某一构念与特定语料库，答案是否定的。分解仅能恢复AMALIA整体性能约一半的水平，错误分析表明模型依赖表面特征关联，尤其是权威人物附近的道德愤怒情绪。而一个开源多语言大语言模型在相同指令下处理同一葡萄牙语语料库时，却能缩小这一差距，说明语料库并非主要解释因素。AMALIA仍可进行大规模筛选与预编码，但尚不足以独立完成对该构念的精准测量。本研究仅为一个反例，并非对国家模型的最终定论；其论证在于：主权大语言模型基准测试不仅应检验与人类编码者的一致性，更应验证达成这种一致性的证据路径。

### 3. [铭记关键时刻：面向长周期任务的主动记忆智能体](http://arxiv.org/abs/2607.08716v1)
- **摘要**: 在长周期任务中，与决策相关的状态信息往往分散在不断扩展的轨迹中，而行动主体必须从中提取关键信息并采取行动。随着轨迹增长，任务要求、环境事实、先前尝试、诊断结果及未完成的子目标可能被埋没在上下文窗口中，甚至被推至窗口之外，导致在需要时无法影响决策。我们将这种失效模式称为"行为状态衰减"。本研究将记忆视为主动干预机制而非被动检索工具。我们设计了一个独立的记忆主体，与未经修改的行动主体并行运行，从近期轨迹中更新结构化记忆库，并决定是否注入基于记忆的提示或保持静默。该模块可与前沿行动主体及现有主体框架实现即插即用。在Terminal-Bench 2.0和τ²-Bench基准测试中，该方法使弱行动主体与强行动主体的pass@1指标均获得提升，在Terminal-Bench上提升8.3个百分点，在τ²-Bench上提升6.8个百分点。消融实验表明，选择性干预优于被动记忆库暴露、持续注入、顾问式引导及通用检索方法。作为开放权重记忆策略的初步探索，我们采用SFT和GRPO方法在SETA数据集上训练Qwen3.5-27B模型，在提升验证奖励的同时，实现了向Terminal-Bench的部分迁移能力。
