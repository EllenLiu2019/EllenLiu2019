### 1. [AI聊天机器人中的广告？大型语言模型如何应对利益冲突的分析](http://arxiv.org/abs/2604.08525v1)
- **摘要**: 当前的大型语言模型（LLMs）通过强化学习等方法被训练以适应用户偏好。然而，这些模型的部署目的已不再仅仅是满足用户需求，还开始通过广告为开发公司创造收入。这可能导致LLMs面临利益冲突——对用户最有利的回应未必符合公司的商业利益。例如，当一款赞助产品价格更高但功能与竞品相当时，LLM应当（或实际会）向用户推荐什么？本文借鉴语言学和广告监管领域的文献，构建了一个分类框架，用以解析利益冲突如何改变LLMs与用户的交互方式。我们随后通过一系列评估实验，检验当前模型如何处理这类权衡问题。研究发现，在多数利益冲突情境下，主流LLMs会牺牲用户利益以迎合公司诉求，具体表现为：推荐价格高出近两倍的赞助产品（Grok 4.1 Fast，83%）、在购买流程中插入赞助选项以干扰决策（GPT 5.1，94%）、在不利比较中隐藏价格信息（Qwen 3 Next，24%）。这些行为还随模型推理能力和用户推断社会经济地位的不同呈现显著差异。我们的研究结果揭示了当企业开始在聊天机器人中隐性激励广告投放时，可能对用户产生的潜在风险。

### 2. [是什么驱动了表征引导？一项关于引导拒绝的机制性案例研究](http://arxiv.org/abs/2604.08524v1)
- **摘要**: 将导向向量应用于大语言模型是一种高效且有效的模型对齐技术，但我们缺乏对其工作原理的可解释性说明——具体而言，导向向量影响了哪些内部机制，以及这种影响如何导致不同的模型输出。为探究导向向量有效性的因果机制，我们以拒绝行为为例展开全面案例研究。我们提出了一种多令牌激活修补框架，并发现不同导向方法在相同层级应用时会利用功能可互换的电路。这些电路表明，导向向量主要通过OV电路与注意力机制交互，而基本忽略QK电路——在两种模型系列中，冻结所有注意力分数仅使导向性能下降8.75%。对受导向OV电路的数学分解进一步揭示了语义可解释的概念，即使在导向向量本身不具备可解释性的情况下亦然。基于激活修补结果，我们证明导向向量可稀疏化90-99%且保持大部分性能，且不同导向方法在重要维度子集上具有一致性。

### 3. [ClawBench：AI智能体能否胜任日常在线任务？](http://arxiv.org/abs/2604.08523v1)
- **摘要**: AI智能体或许能帮你自动处理收件箱，但能否自动化生活中的其他日常事务？日常在线任务为评估下一代AI智能体提供了一个现实且尚未解决的测试平台。为此，我们推出了ClawBench评估框架，包含153项人们在生活与工作中需要定期完成的简单任务，覆盖15个类别中的144个实时平台，涵盖从完成购物、预约服务到提交求职申请等场景。这些任务对AI能力提出了超越现有基准的要求，例如从用户提供的文档中获取相关信息、在多样化平台间导航多步骤工作流，以及填写大量细节表单等重度文本操作。与现有在静态页面离线沙箱中评估智能体的基准不同，ClawBench直接在生产网站上运行，完整保留了真实网络交互的复杂性、动态特性和挑战性。通过轻量级拦截层仅捕获并阻断最终提交请求，确保评估过程安全且不会产生现实副作用。我们对7个前沿模型的评估显示，无论是专有模型还是开源模型，目前都只能完成其中少量任务。例如Claude Sonnet 4.6仅达成33.3%的成功率。ClawBench的进展让我们离实现能作为可靠通用助手的AI智能体更近了一步。
