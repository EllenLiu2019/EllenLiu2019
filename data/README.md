# Data Directory

This directory stores daily ArXiv paper summaries.

## Directory Structure

- `en/`: English papers
- `zh/`: Chinese papers (translated)

## File Naming

Files are named with the format: `YYYYMMDD.md` (e.g., `20251202.md`)

## File Contents

Each file contains:
- Paper title
- ArXiv link
- Publication date
- Full abstract/summary
